# project\_gemu
Bullet hell shooter made with SFML!

### Building the game!
Make sure that libsfml package is installed!  
```
sudo apt install libsfml-dev
```
Build the game using make

```
$ make
```
Finally run the game 
```
$ ./gemu
 ```
