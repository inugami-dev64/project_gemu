# project_gemu
#### Bullet hell shooter made with SFML! <br>
<br>

### Running the game!
In Linux based systems use make to compile the program
```
$ make
```
Then export SFML library path with
```
$ export LD_LIBRARY_PATH=SFML/lib
```
Finally run the game 
```
$ ./gemu
 ```
